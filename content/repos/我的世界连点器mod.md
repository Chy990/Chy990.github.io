# Minecraft 1.21.x 自动连点器 Mod

### GitHub 仓库

[Chy990/20250721_mc_fabric_mod](https://github.com/Chy990/20250721_mc_fabric_mod)

### 简介

这是一个面向 Minecraft 1.21.x 的 Fabric 自动连点器 Mod，主要用于帮助玩家更稳定地挂机刷怪。

与完全自动化的掉落物获取方式相比，它会让玩家使用自己手中的武器进行攻击，因此掉落结果可以正常继承抢夺等武器附魔效果，整体收益更高，也更贴近原版战斗逻辑。

### 主要特点

- 支持 Minecraft 1.21.x 和 Fabric 环境。
- 使用玩家当前持有的武器完成攻击。
- 掉落物可以继承抢夺等武器附魔加成。
- 适合刷怪塔、经验农场等需要持续点击的场景。
