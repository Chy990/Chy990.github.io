# 模板更新历史

## 2026-07-03

- 增强 Markdown 详情页渲染，支持链接、裸 URL、粗体、斜体和行内代码。
- 代码块支持标准三反引号、双反引号，以及 `text`、`bash` 等语言标记。
- 为阅读页链接补充可点击样式，提升 GitHub 仓库地址等内容的可读性。
- 更新静态资源缓存版本号，确保浏览器能加载最新的脚本和样式。

## 2026-07-02

- 一级菜单调整为独立小按钮：笔记、仓库、相册、个人。
- 页面主题调整为浅色优先，深色模式下自动切换为深色背景。
- 移除背景方格纹理，保留鼠标跟随的柔和渐变光晕。
- 增强卡片和按钮 hover 反馈，包括上浮、放大、边框和阴影变化。
- 内容详情页改为全页阅读模式，点开内容后只展示当前文章或图片。
- 优化内容进入和返回动画，改为干净的玻璃面板形变，避免文字压缩和重影。
- 调整主页 Hero 和 Notes 之间的距离，让首页更紧凑。
- 新增自动内容索引脚本 `generate-content-index.py`，减少手动维护 JSON 的成本。

## 2026-06-27

- 搭建静态个人 blog 模板。
- 添加背景粒子动画、滚动入场动画和玻璃质感卡片。
- 建立 Markdown 读取和相册展示的基础逻辑。
